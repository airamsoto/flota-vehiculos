Plataforma de movilidad sostenible

Esta aplicación web permite la gestión eficiente de una flota de vehículos eléctricos, facilitando la reserva, control y mantenimiento desde un entorno accesible.

Beneficios de los vehiculos electricos

  1. Ahorro de combustible
  2. Menos contaminacion
  3. Menos contaminacion acustica

Servicios principales de la plataforma

  • Reserva de vehiculos
  • Estadisticas
  • JOSE

Definiciones calve

HTML

CSS
Hojas de estilo en cascada y se utiliza para dar estilo y diseño a esas páginas
JSON
Es un formato de texto ligero para el intercambio de datos entre sistemas o aplicaciones
